# FUTURE_FS_02
# Simple CRM Application

## 📌 Description
A simple CRM system to manage client leads from website contact forms.

## 🚀 Features
- Add, view, update, delete leads
- Status tracking (New → Contacted → Converted)
- Notes and source tracking
- Dark mode UI with 3D effects

## 🛠 Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB

## ⚙️ How to Run

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm start
